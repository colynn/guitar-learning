# guitar-learning
Learning the guitar


## 注意事项
1. 手势问题
2. 节拍节奏（练琴一定要开着节拍器）
3. 大横按

## 学习阶段
### 基础练习
1. 识谱、调音
2. 节奏（1/4, 3/4, 4/4）
3. 左右手配合
4. 爬格子
5. c调的常见和弦

### 进阶练习
1. 找一些自己喜欢的曲子（谱子）来练习，越简单的越好，这样可以不断增加你的自信心；
2. 针对性的练习一些技巧，比如击勾弦、推弦等；
3. 参照 元子老师 的bilibili课程

在这个过程中 也会吸收一些到乐理知识

### 乐理

1. 乐理
音阶、音程、和弦结构、调式等， 理解很重要，最主要的多思考、多沟通；

2. 乐理与指板结合
